2. Description of internship

 

Fab-lab Rwanda was established in May 2016, by ICT Chamber in partnership with Rwanda Development Board (RDB), Japan International Cooperation Agency (JICA), Ministry of Education,Solidworks Corporation MIT-CBA, and Gasabo 3D who followed through taking lessons learnt from efforts of supporting Software Innovators (at KLAB) to execute on His Excellency’s recommendation to put in place a Fabrication Laboratory (FABLAB) to support Hardware Innovators in Rwanda.

Fab-Lab Rwanda is a space for members to turn innovative ideas into products specifically in the hardware and electronics domain. The space provides a platform for capacity building and integration of hardware skills with software knowledge and quickly bringing Rwandan innovators closer to the Internet of Things era.

2.2 .Equipment used in this internship

In this internship many materials used were:

Computer: was used in daily activities such as: researching, online courses, programming and

Designing .

Eagle software: eagle is a software which used to design pcb boards

Arduino ide : arduino integrated development environment is a software used to which is used to write and upload the computer code to physical board

Arduino board: is a micro controller which can be programmed using arduino ide

Bread board: bread board is solderless board which is used testing of project by put in component  to the board

Soldering iron: soldering iron is machine used to solder the component on pcb board

Mono fab: mono fab is a machine used to design the pcb board by making connection of component to component and make their holes.

Ultimaker 3D printer: ultimaker 3D printer is a machine used to design the plastic which we use to cover the finished project

 

Solidworks (software): solidworks is a software which we use to design the 3d thing (plastics)

Leds: light emitting diode(led) is a two lead semiconductor

Light source it is a p-n junction diode that emits light when activated.

 

Fan: is a machine used to create flow within a fluid ,typically a gas such as air.

Potentiometer: potentiometer is a three terminal resistor with a sliding or rotating contact

That forms an adjustable voltage divider .

 

Jumper wires: jumper wires are used for making connection between items on breadboard and arduino ‘s header pins. Use them to wire up all my circuit

Light depending resistor(ldr): an ldr is a component that has a variable resistance that changes with the light intensity that falls on it . this allows them to be used in light sensing circuit.

Ishigo jam(open source): ishigo jam is a tiny and affordable computer that you can use to learn Programming through fun practical project

Vpanel(software): vpanel is a software used to design set the monofab machine how it will design the pcb board

Circuit.io(website): circuit.io is an online,browser based schematic drawings,pcb design and breadboard simulation package by AutoDesk.
